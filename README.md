# 📖 Project Overview

The **Employee Record System** is designed to manage employee data efficiently in a digital environment. The application provides separate dashboards and functionalities for:

- 👑 Admin
- 🧑‍💼 HR
- 👨‍💻 Employee

The system automates:
- Employee Record Management
- Attendance Tracking
- Performance Evaluation
- Salary Calculation
- Leave & Promotion Requests

while integrating DSA concepts for optimized operations.

# 🧠 DSA Integration

This project demonstrates practical implementation of Data Structures & Algorithms.

| DSA Concept | Purpose |
|---|---|
| List<Employee> | Dynamic Employee Storage |
| Dictionary<ID, Employee> | Fast Employee Lookup |
| Linear Search | Search by Name |
| Binary Search | Efficient Sorted Searching |
| Bubble Sort | Sorting Employee Records |
| Selection Sort | Salary Sorting |
| Max Heap | Top Performer Ranking |
| Queue | FIFO Request Processing |
